## Dynamic Module

`Dynamic Module` 是一个动态组件，它可以根据传入的`JSON Schema`数据动态生成组件。

为了让其可以更好的使用，我们提供了一些基础类型，以及一些常用的组件。

同时为了可以让其可以扩展更多的组件或只需要部分组件，我们提供了简单的解决方案。

由于方便管理数据内容，将使用 `Provider/Inject` 的方式来传递数据。
